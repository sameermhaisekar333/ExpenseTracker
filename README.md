# ExpenseTracker
ExpenseTracker -Flask

## Setup
- Create a new virtualenv named "env"
	>  python3 -m venv env
-  Activate the virtualenv
	- (OS X & Linux)
		> source env/bin/activate
	- (Windows)
		> env\Scripts\activate
- Installing dependencies
	> pip install -r requirements.txt
- Run
	> - python3 run.py
